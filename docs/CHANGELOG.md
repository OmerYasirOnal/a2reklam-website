# Changelog

## [1.0.0] - 2025-03-20

### Added
- Initial release of the A2 Reklam Astro Rebuild project, featuring a modern multilingual static-first website for a2reklam.com, built with Astro 5.

- Updated the build process to utilize `pnpm` as the package manager with optimized scripts for development and production.
- Improved the security measures for repository management, specifically blocking private assets from being tracked.

### Deprecated
- None

- Removed support for legacy service models that did not comply with modern design principles.

### Fixed
- Addressed issues with the contact form email integration, ensuring reliable submission and validation.
- Fixed various bugs related to responsive design, improving user experience across devices.

### Security
- Implemented a safety check in the repository to prevent tracking of private assets.
- Enhanced security features in the contact form, including rate limiting and input validation.
