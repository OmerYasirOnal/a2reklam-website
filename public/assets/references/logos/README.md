# Reference logos

Store optimized client logos here in `.svg` (preferred), `.webp`, or `.png` format. Keep transparent backgrounds and target widths of 200–320px for consistency. Use `.gif` only if an official asset is unavailable.

File naming suggestion: use kebab-case matching the display name (e.g., `pidem.svg`, `ajans-press.svg`). Update `logo` fields in `src/data/references.ts` after adding each file.

See `public/assets/references/logos/SOURCES.md` for the download origins of the current assets.
